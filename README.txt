To begin, run `source scripts/setup.sh` and then `source scripts/runtest.sh`. 
The program will detach and begin to run in a background process. 
To keep up to date with the program, run `tail -f logfile`.
To clean the project directory, run `source scripts/clean.sh`.
Make sure to clean the project directory when switching corpora.
To resplit the corpus, run `python scripts/split.py`.
